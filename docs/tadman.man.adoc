== NAME

tadman - A portable package manager

== Synopsis

*tadman* [OPTIONS]... PATH

== Options

 --dump 		Print out info (for debugging purposes)

 -n		Set the name of the package to be installed manually

 --remove 	Remove the previously generated list of options
